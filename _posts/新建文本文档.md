cf的内网穿透教程
准备工作：
1. 一个cloudflare.com的账号，账号内，有一个域名
2.一个海思机顶盒nas系统
开始部署：
1.下载程序到盒子里：
wget https://git.histb.com/cloudflare/cloudflared/releases/download/2022.4.1（几号就写几)/cloudflared-linux-arm -O /usr/bin/cloudflared
2.授权文件夹后才能登录
chmod +x /usr/bin/cloudflared
3.输入
cloudflared login
登录并获取自动下载的授权文件
4.创建一个隧道
例如
cloudflared tunnel create abc（随便三个字母数字）
得到隧道的id
5.自制配置文件config.yml
例如：
tunnel: 275fe7bb-5948-45a0-bda7-30da7d154ddf（替换为上边的id)
credentials-file: /root/.cloudflared/275fe7bb-5948-45a0-bda7-30da7d154ddf.json（替换为上边的id）
ingress:
- hostname: abc.slitaz.cf（替换为自己的）
service: http://127.0.0.1
- service: http_status:404
6.创建一个文件夹
mkdir /etc/cloudflared/
把这个配置文档放置到/etc/cloudflared/下面
vi /etc/cloudflared/config.yml（记得先按i进入编辑模式）
7.运行一下这个隧道
cloudflared tunnel route dns web myweb.yueyuebook.tk（替换为自己的）
8.运行自启动安装
cloudflared service install
systemctl start cloudflared
systemctl status cloudflared
搞定！
ps: service: http://127.0.0.1 这个记得按完tab键后,还有2个空格键
